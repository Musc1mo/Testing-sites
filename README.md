# English School Website

Простой веб-сайт для школы английского языка.

## Описание

Это одностраничный сайт (landing page) для школы английского языка, созданный на чистом HTML и CSS. Сайт включает следующие разделы:

- **Главная страница** - приветственный баннер с призывом к действию
- **О нас** - информация о преимуществах школы
- **Курсы** - описание доступных программ обучения
- **Преподаватели** - информация о квалифицированных учителях
- **Контакты** - форма обратной связи и контактная информация

## Технологии

- HTML5
- CSS3 (с использованием Flexbox и Grid)
- Адаптивный дизайн (responsive design)

## Как открыть

1. Откройте файл `index.html` в любом современном браузере
2. Или запустите локальный веб-сервер в директории проекта

### Запуск локального сервера

С Python 3:
```bash
python -m http.server 8000
```

С Node.js (если установлен npx):
```bash
npx http-server
```

После запуска сервера откройте в браузере: `http://localhost:8000`
